# StreamTyper

[English](README.md) | [中文](README_zh-CN.md)

StreamTyper 是一个 macOS 实用工具，它可以模拟键盘输入来逐字打出剪贴板中的文本。这对于将文本粘贴到阻止标准粘贴操作的应用程序中，或用于自动化文本输入非常有用。

## 功能特性

- **模拟打字**：逐个字符地输入文本。
- **速度控制**：调整打字速度以适应目标应用程序的响应速度。
- **剪贴板槽位**：管理多个文本片段。
- **换行模式**：配置打字时如何处理换行符。
- **全局热键**：使用可自定义的键盘快捷键触发打字。

## 系统要求

- macOS 11.0+
- 辅助功能权限（模拟按键所需）

## 安装

1. 克隆仓库。
2. 在 Xcode 中打开 `TypeStream.xcodeproj`。
3. 构建并运行。
4. 根据提示授予辅助功能权限。

## 使用方法

1. 将文本复制到剪贴板。
2. 使用配置的热键开始打字。
3. 根据需要在主窗口中调整设置。
