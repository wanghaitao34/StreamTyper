# StreamTyper

[English](README.md) | [中文](README_zh-CN.md)

StreamTyper is a macOS utility that simulates keyboard input to type out text from your clipboard. It is useful for pasting text into applications that block standard paste operations or for automating text entry.

## Features

- **Simulated Typing**: Types text character by character.
- **Speed Control**: Adjust the typing speed to match the target application's responsiveness.
- **Clipboard Slots**: Manage multiple text snippets.
- **Newline Modes**: Configure how newlines are handled during typing.
- **Global Hotkeys**: Trigger typing with customizable keyboard shortcuts.

## Requirements

- macOS 11.0+
- Accessibility Permissions (required to simulate key presses)

## Installation

1. Clone the repository.
2. Open `TypeStream.xcodeproj` in Xcode.
3. Build and run.
4. Grant Accessibility permissions when prompted.

## Usage

1. Copy text to your clipboard.
2. Use the configured hotkey to start typing.
3. Adjust settings in the main window as needed.
