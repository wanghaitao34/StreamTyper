### English Version

StreamTyper is a clipboard management and auto-typing tool.

**Accessibility Permission Explanation:**
This app requires accessibility permission to simulate keyboard input. This is the core functionality that allows automatic typing of clipboard content into any application.

**Testing Steps:**
1. Launch StreamTyper
2. Grant accessibility permission in System Settings > Privacy & Security > Accessibility
3. Copy text to clipboard (e.g., "Test text")
4. Open any text editor (e.g., TextEdit, Notes)
5. Click on Slot 1 in StreamTyper to load the text
6. In the text editor, press Cmd+Shift+Option+1 to trigger auto-typing
7. Observe the text being typed automatically

**Additional Features to Test:**
- Adjust typing speed slider
- Switch newline handling modes
- Switch interface language (top right corner)
- Test all 4 clipboard slots

**Privacy Statement:**
This app works completely offline and does not collect any user data or connect to any servers. All data is stored locally in UserDefaults.

**Technical Implementation:**
- Uses ApplicationServices framework's CGEvent API for keyboard simulation
- Uses Carbon framework for global hotkey registration
- Data storage uses UserDefaults (local only)

If you have any questions, please contact us through App Store Connect.

Thank you!
